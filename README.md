# 📘 Calificaciones  
### Guía práctica de Programación Orientada a Objetos (POO)

Este repositorio ha sido diseñado como material de apoyo para la enseñanza de la materia **Programación Orientada a Objetos**. A través de un caso práctico (sistema de calificaciones), se explican e implementan los conceptos fundamentales del paradigma orientado a objetos.

---

## 🎯 Objetivo

Proporcionar a los estudiantes una base sólida para comprender y aplicar los principios de la POO mediante un ejemplo realista, progresivo y modificable.

---

## 🧠 Conceptos abordados

Este proyecto permite estudiar e identificar en código los siguientes conceptos clave:

- 🧱 Clases y Objetos  
- 🏷️ Atributos y Métodos  
- 🔒 Encapsulamiento  
- 🎭 Abstracción  
- 🧬 Herencia  
- 🔁 Polimorfismo  
- ⚙️ Constructores  

---

## 🧩 Caso de estudio: Sistema de Calificaciones

El sistema simulara un entorno académico básico donde se pueden:

- Registrar estudiantes  
- Asignar calificaciones  
- Calcular promedios  
- Mostrar resultados  

Este enfoque permite modelar entidades del mundo real utilizando POO.

---

## 🛠️ Tecnologías utilizadas

- Lenguaje: C# (o el que estés usando)
- Paradigma: Programación Orientada a Objetos
- Entorno: .NET / Consola

---

## 👨‍🏫 Uso en clase

Este repositorio está pensado para:

- Explicar conceptos paso a paso  
- Realizar prácticas guiadas  
- Proponer ejercicios basados en el código  
- Evaluar comprensión mediante modificaciones  

---

## 🧪 Ejercicios sugeridos

Para reforzar el aprendizaje, intenta:

- Agregar validaciones a las calificaciones  
- Implementar diferentes tipos de evaluación  
- Crear una clase `Curso`  
- Aplicar herencia (ej. `Persona → Estudiante`)  
- Implementar interfaces  

---

## 📌 Recomendaciones para estudiantes

- No solo ejecutes el código → **entiéndelo**
- Relaciona cada clase con un concepto de POO
- Experimenta cambiando la lógica
- Rompe el código y arréglalo (aprendizaje real)
